# toDoApp API

This repository contains toDoApp API 

## Features

- Create List
- Read List
- Read List by ID
- Update List
- Delete List

## Oncoming Feature
- Authentication

## Installation

_Note_: You will need MongoDB running locally on your machine.

1. Clone this repository
2. Run `npm install`
3. Run `npm run start:dev`
4. Server will connect to MongoDB & run on Port 3001.
